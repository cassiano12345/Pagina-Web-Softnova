### Projeto pagina web softnova

Este projeto teve como objetivo criar uma pagina web para a empresa softnova e os clientes. Para tal foi usado PhP, HTML, Css, Javascript, e base de dados em Oracle num servidor.

### Algumas funcionalidades a destacar
***Pasta -> bin -> config.php*** <br/>

No ficheiro config.php encontramos a conexão a base de dados. 

***Pasta -> Modules*** <br/>

Na pasta Modules encontramos todas as funcionalidades e os atores da pagina web, bem como as consultas a base de dados para cada pagina.

***Pasta -> Modules -> Main ->index.php*** <br/>
O ficheiro index.php na pasta main é praticamente o centro da pasta e é la que encontra se todo o codigo Javascript que faz funcionar a pagina e dinamiza a pagina, e ele que faz mudar de pagina, adiciona botões a barra lateral, etc.
